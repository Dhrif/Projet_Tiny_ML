## Project Overview

This project implements an **embedded image classification model** designed to recognize different electronic components using images captured by a camera connected to an embedded device.

### Project Details

- **Project Name:** Classification_Composants   
- **Input Resolution:** 64 × 64 pixels (grayscale)  
- **Classes:**  
  - Background  
  - LED  
  - DHT11 sensor  
- **Target Hardware:** Arduino Nano 33 BLE Sense  

The model is optimized to run directly on a **microcontroller with limited memory and processing power**, allowing real-time inference on the embedded device.

---

## What You Can Do with This Link

Accessing this project allows you to:

-  Explore the **training dataset**
-  Inspect the **model architecture and impulse configuration**
-  Review **training metrics and model performance**
-  Test the model using the **Live Classification** tool
-  Export the **Arduino inference library** for deployment
